# AI Training — Materials Hub

## 定位

SmartTextbook 在这里的角色是**材料制作工具**：把课程内容快速转化为可用的教学材料（互动课件、AI辅导对话、测验题组），而不是替代教师的教学过程。

AI培训工作坊是第一个落地场景，但这个文件夹长期可服务于任何课程。

---

## 文件夹结构

```
ai-training/
├── README.md               ← 本文件（定位与概览）
├── workshop-materials/     ← 工作坊用的培训材料
│   └── ...
├── course-specific/        ← 各课程的专属材料
│   └── ...
└── resources/              ← 通用资源（提示词模板、评分标准等）
    └── ...
```
