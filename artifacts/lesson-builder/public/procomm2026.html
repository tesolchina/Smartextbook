About This Workshop

What You'll Walk Away With

In 75 minutes, you'll produce a complete interactive learning module from a real ProComm article — using only natural language instructions to an AI agent.

A structured 4-stage workflow for transforming any static article into an interactive lesson. Reusable across topics and formats.

Research Context

The Framework

The PITA Workflow

Four stages. Natural language at every step. No code required.

P
Parse
AI reads the article, identifies key concepts, and maps the pedagogical structure
I
Identify
You choose which concepts need active practice — scenarios, quizzes, reflection prompts
T
Transform
AI assembles a single-file HTML interactive module, no server required
A
Augment
Add audio narration, design annotations, and xAPI learning analytics
