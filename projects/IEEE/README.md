# IEEE ProComm — Project Hub

## 项目概览

**核心目标**: 把 IEEE ProComm 网站上 127 篇静态专业沟通文章，用 Agentic AI（vibe coding）转化为互动学习页面

**主要联系人**:
- **Traci Nathans-Kelly** (IEEE ProComm VP, Content) — tracink.ieee@gmail.com
- 由 George Hayhoe（前编辑）介绍引荐

---

## 两条并行轨道

### 轨道 A — ProComm 2026 工作坊
- **会议**: IEEE ProComm 2026，Edmonton, Canada，2026年7月12–15日
- **主题**: From Static to Interactive — PITA 框架 (Parse → Identify → Transform → Augment)
- **形式**: 75分钟工作坊，无需编程经验
- **Camera-ready 截止**: 2026年4月30日

### 轨道 B — IEEE TPC Teaching Case
- **目标期刊**: IEEE Transactions on Professional Communication
- **题目**: "Teaching Agentic AI Tool Use to Non-Programming Students"
- **编辑**: Dr Craig Baehr — stcbaehr@gmail.com（已正面回应）
- **目标投稿**: 2026年6月
